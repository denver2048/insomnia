# Architecture Decision Records (ADR)

This folder contains Architecture Decision Records for the Insomnia project (Intelligent Night Operations Monitoring & Investigation Agent).

## Index

| ADR   | Title                                               | Status   |
|-------|-----------------------------------------------------|----------|
| 0001  | [Insomnia project architecture](0001-insomnia-architecture.md) | Accepted |

## Format

Each ADR includes:

- **Status**: e.g. Accepted, Deprecated, Superseded
- **Context**: What situation or problem led to this decision
- **Decision**: What we decided
- **Consequences**: Benefits, drawbacks, and operational notes

New ADRs should be numbered sequentially (e.g. `0004-...`) and added to this index.
